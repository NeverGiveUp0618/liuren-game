起源：黄帝战蚩尤，风后作六壬；或传九天玄女授黄帝。汉代出土式盘佐证，六壬为汉前学术。六壬最先出，奇门次之，太乙晚出。
三式对比：太乙主天道国运，奇门主地道兵事，六壬主人道人事。六壬唐代未被禁传，流传最广，誉为"人事之王""占卜之王"。
历史沿革：唐代盛行 → 宋代完善（邵彦和、凌福之）→ 明清郭御青辑《六壬大全》入《四库全书》。
核心逻辑：月将加时 → 排天地盘 → 立四课 → 发三传 → 配天将神煞 → 断吉凶。
天文依据：月将即太阳，随中气换宫，主宰天盘。古云"月将为误，然后凡事不误"。式盘：天盘（圆，动）嵌地盘（方，静）。
四课三传：四课体现日干支阴阳神关系（两仪生四象）；三传（初、中、末）示事态始、中、终。九宗门（贼克·比用·涉害·遥克·昴星·别责·八专·伏吟·返吟）。
神将体系：十二天将赋予地支社会属性。贵人分昼夜，昼贵顺行，夜贵逆行，主宰吉凶。
核心书目：《大六壬心镜》《大六壬断案》（邵彦和）、《六壬大全》（郭御青）、《六壬毕法赋》（凌福之）、《壬归》、《大六壬会纂占验指南》。
起课流程：定月将 → 排地盘 → 月将加时起天盘 → 排十二天将 → 起四课 → 定三传 → 配遁干 → 列六亲。
判断流程：审格局 → 析四课 → 推三传 → 看六亲类神 → 查神煞空亡 → 参本命行年。
心法三要：①类神思维（凡有其象必有其事）②旺相休囚（凶神吉将不可一例而推）③阴阳辩证（日干为阳我，日支为阴彼）。
太岁：年柱，司一年吉凶。月建：月柱，主节气旺衰。先锋门：时柱，断课切入点。本命：出生年干支，为体。行年：流年，男顺女逆，为用。
四大断则：①生克为体（日干为我，生我吉克我凶）②旺相为用（旺相力大，休囚力微，空亡不成）③动静迟速（驿马主动速，伏吟墓合主静迟）④去伪存真（聚焦天将三传，活断结合情境）。
阳干：甲丙戊庚壬；阴干：乙丁己辛癸。阳支：子寅辰午申戌；阴支：丑卯巳未酉亥。天干为阳主动，地支为阴主静。
五行属性：木（甲乙寅卯）火（丙丁巳午）土（戊己辰戌丑未）金（庚辛申酉）水（壬癸亥子）。顺生逆克。
旺相休囚死（以月令定）：当令为旺，我生为相，生我为休，克我为囚，我克为死。
纳音五行：六十甲子每两组配一纳音（如甲子乙丑海中金）。干主禄，支主命，纳音主身/寿。高层断课综合三者生克。
节气换月：六壬以"节"为月令界限（非农历初一）。立春后寅月，惊蛰后卯月……小寒后丑月。
月建：正月建寅，顺推至十二月建丑。月将（太阳）取中气后太阳所躔之宫，与月建相冲。

空亡断诀：空亡主虚无落空。吉神空亡减半，凶神空亡解灾或无力。需察填实、冲实之应。

式盘：地盘（方，千古不变，刻十二地支）+ 天盘（圆，随月将加时顺行）+ 人盘（十二天将，随贵人顺逆排布）。
十二天将顺序：贵（天乙）蛇（螣蛇）雀（朱雀）合（六合）勾（勾陈）龙（青龙）空（天空）虎（白虎）常（太常）玄（玄武）阴（太阴）后（天后）。
贵人起法：昼贵（卯至申时）顺行；夜贵（酉至寅时）逆行。贵人在亥子丑寅卯辰顺行，在巳午未申酉戌逆行。
四课含义：第一课（日阳）= 日干上神；第二课（日阴）= 第一课上神之地盘上神；第三课（辰阳）= 日支上神；第四课（辰阴）= 第三课上神之地盘上神。
记忆：一二课看天（干），三四课看地（支）。上克下为"克"，下克上为"贼"。
①克贼法：先取下克上（贼），无贼取上克下（克）。一上克下→元首课；一下克上→重审课。
②比用法：多课相克，取与日干阴阳相比者发用。二下贼→比用；二上克→知一。
③涉害法：多克阴阳不比，顺数至本家统计克害，取深者。相等则：有孟取孟（见机）→ 有仲取仲（察微）→ 复等：阳日取干上、阴日取辰上（复等）。
④遥克法：四课无正克。日干遥克上神→蒿矢（取初传）；上神遥克日干→弹射。
⑤昴星法：无克无遥克。阳日取冲时之支（天门）上神，阴日取地户上神。
⑥别责法：柔日无克无遥克，二三课同。初传取日支前三合，中末均取日上神。
⑦八专法：干支同位，仅两课。有克按克法；无克刚日顺数、柔日逆数三位发用，中末取日上神。
⑧伏吟法：天地盘重合。有克按克贼；无克阳日取日干刑神，阴日取日支刑神。刑序：寅巳申、丑戌未、子卯、辰午酉亥自刑。
⑨返吟法：天地盘对冲。有克按克贼；无克（井栏射）取日驿马，中传取支上神，末传取干上神。
六亲（以日干为我）：生我→父母，我生→子孙，克我→官鬼，我克→妻财，同我→兄弟。
三传：初传为事始，中传为事变，末传为事终。
伏吟：主静、迟、内忧。自任格（阳日）宜静，自信格（阴日）潜藏。
返吟：主动、速、反复。无依格主祸自外来，无亲格主速成易破。
阳日顺行：贵→蛇→雀→合→勾→龙→空→虎→常→玄→阴→后。阴日逆行（顺序反转）。
昼贵（卯至申）：甲戊庚→丑，乙己→子，丙丁→亥，辛→午，壬癸→巳。
夜贵（酉至寅）：甲戊庚→未，乙己→申，丙丁→酉，辛→寅，壬癸→卯。
贵人在亥子丑寅卯辰→顺行；在巳午未申酉戌→逆行。原理：亥为天门，巳为地户。

关键节点：子午为天关（阴阳交替）；卯酉为日月门（门户变动）；寅申为道路神（主出行）；辰戌为牢狱地（主刑狱）；丑未为天厨（主富贵）；巳亥为堂庙（主征召）。
天罡（辰）加干：主变动官非。河魁（戌）：主终结坟墓。功曹（寅）与传送（申）逢冲主动，逢合主止。神后（子）逢玄武主盗，逢天后主婚。
地形地貌：亥子江湖，寅卯山林，丑田，未井，午市，巳窑，申园场，戌营寨，辰衙庭，酉城郭。
基本逻辑：支神为体（事类），天将为用（吉凶）。吉神吉将相生则成，凶神凶将无克战则平。
青龙+寅卯：升迁文书吉。白虎+申酉：血光官司丧。玄武+子亥：盗贼遗失。朱雀+巳午：口舌火灾。勾陈+辰戌：田土牢狱。螣蛇+巳寅：惊恐虚诈。天后+子未：婚姻阴私。六合+卯未：婚姻交易。
取课优先级：克贼（元首/重审）→ 比用 → 涉害 → 遥克（蒿矢/弹射）→ 昴星 → 别责 → 八专 → 伏吟 → 返吟。
伏吟：神将归位，主静迟阻滞。刚日自任（干上发），柔日自信（辰上发），自刑为杜传。
返吟：神将对冲，主动反复。有克依贼克，无克（井栏射）驿马发用。
三光格：日德、日禄、日马入传，无刑冲破害。主光明正大，百事亨通。
三阳格：三传皆阳神，或贵人在阳位。主阳气旺盛，利进取求官。
龙德格：青龙乘德神加临或发用。主恩泽降临，仕途顺遂，解诸灾厄。
铸印格：干上克支上（铸印，掌权）；斫轮：支上克干上（受制）。
引从格：三传顺逆相生。合欢格：三传皆合或干支相合，主合作顺利，婚姻和谐。
斩关格：初传为日干冲神，或辰戌加干支。主冲破阻碍，险中求胜，亦主逃亡官非。
闭口格：初传为日干墓神，或逢空墓。主口舌是非，言语受阻，事多隐晦。
天网格：三传皆空，或辰戌加临。天狱格：三传皆刑，或墓神加日。主牢狱虚无。
天寇格：三传皆贼克日干，或初传鬼乘白虎。主盗贼侵扰，意外灾祸。
魄化格：白虎乘死神加临。主死亡血光，六壬最凶之格。
吉凶互见：吉格见白虎玄武则吉中有凶；凶格见青龙贵人则凶中有吉。德神化凶，合神成事。
空亡影响：空亡初传斩首，中传折腰，末传刖足。吉格逢空减半，凶格逢空可免。
动静之机：伏吟主静，返吟主动；斩关主破，闭口主藏。初传定基调，中传看过程，末传定结果。
日干为"我/内/主/男/阳"，日支为"彼/外/客/女/阴"。干上神克支上神→主胜客；支上神克干上神→客胜主。
日上生日→百事吉；日上克日→百不利；日生上神→主耗财；日克上神→事抑塞。
干支交生→"脱体"主信任；交克→"无涉"主猜忌。
三传逻辑：初传为事始（先锋），中传为事变（转折），末传为事终（归宿）。三传顺生→吉，逆克→凶，回头克→始吉终凶。
课体吉凶：元首课（上克下）吉，事从外来；重审课（下贼上）凶，事从内起。伏吟静迟，返吟动荡。
日干神克命上神，今日求财明日成。 命上神克日上神，官符灾讼不康宁。 日上神合命上神，此年喜事自生春。
五行旺衰以月令定。旺者力强，死者力弱。长生帝旺为吉，死墓绝为凶。墓主暗昧（喜冲忌合），绝主断绝（亦主绝处逢生）。
刑冲合害：三合主聚（事成），六冲主散（事败），刑主伤残官非，冲主变动倾覆（凶冲则散），合主和合羁绊（合中带煞），害主阻隔不顺。
类神取用：财取青龙/妻财，官取白虎/官鬼。类神落空、入墓、被克则事不成；得地、逢生、合助则事可成。
虚实聚散：空亡为虚，填实为实；三合六合为聚（主事成），六冲破害为散（主事败）。
动静进退：初传生末传→进（事速成），克末传→退（事迟滞）。天盘动地盘静，初传动末传静。
以类神为核心，结合干支、神将、六亲、神煞综合判断。课传生合为吉，克害冲刑为凶。
空亡断诀：妻财入空先难后成；子孙入空初年难为子；贵人入空求名虚声；凶将入空（蛇虎玄）凶事可免；吉将入空（龙常合）财喜减半。
天时：青龙主晴，太常主雨，朱雀主雷火。亥为雨师，巳为火神。勾陈制玄武主晴，玄武制勾陈主雨。
阳宅：干上神（宅）与支上神（人）。干生支吉，支生干亦吉；干克支主宅逼人，支克干主人逼宅。忌白虎玄武临支。
阴地：看龙德、官爵等吉格。忌白虎死气临支。
婚姻：男看干上神，女看支上神。干支相合、六合、天后临干支为吉；干支相克、玄武螣蛇临之则凶。九丑卦主夫妻反目。
孕产：看子孙及天后。子孙旺相、天后生合则顺产；白虎死气临之则难产。胎坐空不受，遇冲小产。
疾病：白虎克日干主病重；日干生白虎主病退。生死：日干入墓逢空被克无救主死；得贵人德神生合主生。
出行：马星动、干上神生合主行；伏吟、闭口课主不行。行人：干上神克支上神主未归，生合主将归。
趋谒：贵人临干、官星生合主见贵成功；贵人落空被克主不见或无功。
求财：财星生合日干、青龙临财主得财；财星空亡被克主破财。买卖：干生支主卖，支生干主买，干支比和主交易成。
官讼：朱雀主口舌，勾陈主牢狱。官鬼生合日干主胜，克日干主败。失物：玄武临干支主被盗；财星空亡主难寻；得合主可找回。
